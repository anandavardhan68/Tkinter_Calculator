Simple Calculator (Tkinter)
Description

This is a basic desktop calculator built using Python and Tkinter. It provides standard arithmetic operations including addition, subtraction, multiplication, and division. The interface is simple, clean, and user-friendly, designed using Tkinterâ€™s grid layout system.

Features

Graphical user interface built with Tkinter

Supports +, -, *, and / operations

Decimal number support

Clear button to reset input

Backspace button to remove the last character

Error handling for invalid expressions

Technologies Used

Python 3

Tkinter (built-in Python GUI library)

How to Run

Make sure Python 3 is installed on your system.

Save the file as calculator.py.

Open a terminal or command prompt in the project directory.

Run the following command:

python calculator.py


The calculator window will open and is ready to use.
